Objetivo Geral

Desenvolver uma aplicação mobile que permita o registro automático da presença de participantes em sessões de eventos acadêmicos, utilizando QR Code e timers para garantir que o participante permaneceu tempo mínimo suficiente, resolvendo o problema de demora e erro no registro manual de presenças.
